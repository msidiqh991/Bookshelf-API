
# Bookshelf-API

Submission for the last course of the dicoding program website.
## Installation
Clone the project

```bash
  git clone https://github.com/msidiqh991/Bookshelf-API.git
```

Port server
```bash
  Port: 9000
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Server:** Node, Hapi
